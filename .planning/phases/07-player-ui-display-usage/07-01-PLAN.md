Crear directorios StrataClient/Player/Lyrics/ y StrataClient/Player/Chords/.

**LyricsModels.swift:**
- `LyricsFile: Decodable` con `language: String`, `segments: [LyricLine]`
- `LyricLine: Decodable, Identifiable, Equatable` con `id = UUID()` (excluido de CodingKeys), `start: Double`, `end: Double`, `text: String`, `words: [LyricWord]`
- `LyricWord: Decodable, Identifiable, Equatable` con `id = UUID()` (excluido de CodingKeys), `word: String`, `start: Double`, `end: Double`
- Equatable basado en start+text (no UUID) para que `.onChange(of:)` funcione correctamente

**ChordModels.swift:**
- `ChordsFile: Decodable` con `chords: [ChordEntry]`
- `ChordEntry: Decodable, Identifiable, Equatable` con `id = UUID()` (excluido de CodingKeys), `start: Double`, `end: Double?` (null para ultimo acorde — se rellena con duration del engine), `chord: String`
- Equatable basado en start+chord

**ChordTransposer.swift:**
- `enum ChordTransposer` (namespace, no instanciable)
- `static let sharpRoots = ["C","C#","D","D#","E","F","F#","G","G#","A","A#","B"]`
- `static let flatRoots = ["C","Db","D","Eb","E","F","Gb","G","Ab","A","Bb","B"]`
- `static func transpose(_ chord: String, semitones: Int) -> String`: extrae root (hasta 2 chars, busca en allRoots = sharpRoots + flatRoots por hasPrefix, ordenado de mayor longitud primero para que "C#" matchee antes de "C"), aplica modulo 12, elige sharps para semitones > 0, flats para semitones < 0, recombina con suffix
- `static func inferKey(from chords: [ChordEntry]) -> String?`: agrupa por rootOnly(), devuelve el root mas frecuente
- `private static func rootOnly(_ chord: String) -> String`: extrae solo la raiz del acorde (sin m, 7, dim, etc.)

Todos los structs deben ser Sendable (structs con campos value-type lo son implicitamente).
