Crear StrataClient/Player/Lyrics/LyricsView.swift:

```swift
struct LyricsView: View {
    @Environment(PlayerViewModel.self) private var vm
    @Environment(PlaybackEngine.self) private var engine
}
```

**Layout:**
- ScrollViewReader { proxy in ScrollView { LazyVStack(spacing: 16) { ForEach(vm.lyrics) { line in LyricLineView(line: line, ...).id(line.id) } } .padding(.vertical, 200) } }
- El padding vertical grande (200pt) permite que la primera y ultima linea puedan centrarse en pantalla

**Scroll sincronizado:**
- `.onChange(of: vm.currentLine?.id) { _, newID in guard let id = newID else { return }; withAnimation(.easeInOut(duration: 0.3)) { proxy.scrollTo(id, anchor: .center) } }`
- IMPORTANTE: Solo hacer scrollTo cuando cambia la LINEA, no en cada frame de 60fps. currentLine cambia cada pocos segundos.
- Al hacer seek, currentLine cambia inmediatamente → el scroll salta al punto correcto.

**LyricLineView** (subview en mismo archivo):
- `let line: LyricLine`
- `let isActive: Bool`
- `let activeWord: LyricWord?`
- Body: texto construido como concatenacion de Text views con `.bold()` y `.foregroundColor(.accentColor)` en la palabra activa:
  ```swift
  HStack(spacing: 0) { // o un flow layout con Text concatenados
      ForEach(line.words) { word in
          let isHighlighted = (activeWord?.id == word.id)
          Text(word.word + " ")
              .font(.title2)
              .fontWeight(isHighlighted ? .bold : .regular)
              .foregroundColor(isHighlighted ? .accentColor : .primary)
      }
  }
  .frame(maxWidth: .infinity, alignment: .center)
  ```
- Linea no activa: `.opacity(isActive ? 1.0 : 0.4)`
- Pasar isActive y activeWord desde LyricsView: `LyricLineView(line: line, isActive: line.id == vm.currentLine?.id, activeWord: line.id == vm.currentLine?.id ? vm.currentWord : nil)`

**Sin click-to-seek** — per user decision, las lineas son solo visuales.

Nota performance: Las LyricLineView de lineas no activas no necesitan re-renderizar en cada frame. Solo la linea activa cambia frecuentemente (por el highlight de palabra). LazyVStack ayuda.
