Crear StrataClient/Player/PlayerViewModel.swift:

```swift
@Observable
@MainActor
final class PlayerViewModel {
    let song: SongEntry
    private(set) var lyrics: [LyricLine] = []
    private(set) var chords: [ChordEntry] = []
    var showTransposed: Bool = false

    private let engine: PlaybackEngine
    private let cacheManager: CacheManager
    private let libraryStore: LibraryStore

    // Indices para O(1) lookup en caso comun (avance secuencial)
    private var lastLineIndex: Int = 0
    private var lastChordIndex: Int = 0
}
```

**init:** Recibe song, engine, cacheManager, libraryStore. Restaura pitchOffset de song.pitchOffset ?? 0 via engine.setPitch().

**func load() async throws:**
1. Cargar lyrics.json: `let lyricsURL = await cacheManager.lyricsURL(songID: song.id)`. Si existe, decodificar como LyricsFile, asignar a self.lyrics.
2. Cargar chords.json: `let chordsURL = await cacheManager.chordsURL(songID: song.id)`. Si existe, decodificar como ChordsFile, asignar a self.chords. Cerrar el ultimo acorde: si `chords.last?.end == nil`, setear `chords[chords.count - 1].end = engine.duration`.
3. Cargar stems: construir array de URLs [vocals.wav, drums.wav, bass.wav, other.wav] via cacheManager.stemURL(), llamar `try engine.load(stemURLs:)`.
4. Inferir key si song.key == nil: `let key = ChordTransposer.inferKey(from: chords)`. Si key != nil, actualizar song.key en libraryStore (leer indice, actualizar, escribir).

**var currentLine: LyricLine?** (computed):
- Buscar desde lastLineIndex hacia adelante (optimization para avance secuencial). Si engine.currentTime esta entre line.start y line.end, actualizar lastLineIndex y retornar. Si no, busqueda lineal completa como fallback (seek).

**var currentWord: LyricWord?** (computed):
- Si currentLine != nil, buscar en currentLine.words donde word.start <= currentTime < word.end.

**var currentChord: ChordEntry?** (computed):
- Buscar desde lastChordIndex: ultimo acorde donde chord.start <= engine.currentTime. Optimization similar a currentLine.

**var nextChord: ChordEntry?** (computed):
- Si currentChord encontrado, devolver chords[index + 1] si existe.

**var displayChord: String** (computed):
- Si showTransposed && engine.pitchSemitones != 0: ChordTransposer.transpose(currentChord?.chord ?? "", semitones: engine.pitchSemitones)
- Else: currentChord?.chord ?? ""

**var displayNextChord: String** (computed):
- Misma logica de transposicion para nextChord.

**func savePitchOffset() async:**
- Guardar engine.pitchSemitones en song y persistir via libraryStore/cacheManager. Llamar esto cuando el usuario cambie pitch.
